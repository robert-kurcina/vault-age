# Guidance Integration Map

## Local definitions

Amazing Game Engine [AGE] means the complete platform described by this archive.

Large Language Model [LLM] means a generative language model used for language interpretation or presentation.

Large Model System [LMS] means the wider model, retrieval, graph, tool, agent, and workspace system around one or more LLMs.

Large Model System Graph [LMS-Graph] means the maintained graph/relational corpus substrate.

Corpus Arbitration Layer [CAL] means the component that answers unstructured questions against LMS-Graph.

Roles-as-a-Service [RaaS] means a deprecated umbrella label in this archive; use Role Service, Rules Service, CAL, LMS-Graph, and Agent Service instead.

## Purpose

This file records how the Claude-thread guidance summary is incorporated into the revised archive.

## Incorporated guidance

The project is framed as more than a gaming platform, but the gaming platform remains the first safe proof environment.

LMS is defined as a Large Model System, not merely an LLM.

LMS-Graph is defined as a recursively expanded graph/relational corpus substrate built from authoritative roots.

CAL is introduced as the Corpus Arbitration Layer, the central anchored-corpus query and answer component.

Rules Service is narrowed to rules-specific CAL deployment.

Role Service is separated from Rules Service and CAL.

RaaS is deprecated as an umbrella term because it overloaded roles, rules, retrieval, and arbitration.

The human authority model is made explicit across gaming, serious play, and professional domains.

The CAL output envelope is formalized.

Gaming rules arbitration is treated as the first production test bed, not as a metaphor.

Market-selected expansion replaces any premature commitment to a first professional domain.

## Resulting doctrine

AGE retrieves, structures, weighs, simulates, explains, and records. It does not replace final judgment.

The first development goal is not a universal professional knowledge system. The first development goal is a state-authoritative game runtime with a bounded game rules LMS-Graph and CAL-based Rules Service.
