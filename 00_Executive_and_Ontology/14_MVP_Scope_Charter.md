# Amazing Game Engine [AGE] Minimum Viable Product [MVP] Scope Charter

## Local definitions

Amazing Game Engine [AGE] means the complete platform described by this archive.

Corpus Arbitration Layer [CAL] means the component that answers unstructured questions against LMS-Graph.

Minimum Viable Product [MVP] means the smallest product that proves the core AGE loop.

Quality Assurance [QA] means testing and review; Semantic QA tests meaning, authority, role behavior, rules, state, and prose-state agreement.

Application Programming Interface [API] means a software interface used by one system to request services from another.

## Purpose

The MVP exists to prove AGE’s core architecture, not its entire long-term platform. It should demonstrate state authority, natural-language agency, deterministic resolution, scope/time handling, rules arbitration, output verification, replay, and human override capture.

## In scope

- One troupe.
- One bounded game world or adventure package.
- One owned or licensed rules corpus.
- Player natural-language input.
- Structured action candidates.
- Bridge Layer validation.
- Deterministic domain resolution.
- State Mutation Engine.
- Event Bus.
- State Assembler.
- Fiction Constructor contract.
- Output Verifier.
- Replay Log.
- Spatial scope through at least submap and locus.
- Narrative scope through at least spotlight, scene, and act.
- Temporal scope through moment, scene time, travel, and downtime compression.
- AGEScript minimum viable schema.
- Lazy ontology actualization for observed/needed details.
- CAL quick ruling and detailed ruling paths.
- Referee override capture.
- Semantic QA for state contradiction and invalid rulings.

## Out of scope

- Professional-domain deployment.
- Agent Service/API action.
- Unrestricted marketplace.
- Unbounded multiplayer at MMO scale.
- Fully simulated world economy.
- Full legal, medical, building code, or regulated corpus ingestion.
- Autonomous final decision-making.
- Certification as an external standard.

## MVP success statement

A player can complete a bounded adventure through natural language while AGE preserves state, time, place, rules, consequences, corpus-grounded rulings, replay, and human authority.
