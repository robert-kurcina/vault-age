# Amazing Game Engine [AGE] Revised Development Archive - Definition-First Pass

Amazing Game Engine [AGE] means the total platform. AGE Engine means the deterministic runtime. Large Model System Graph [LMS-Graph] means the maintained graph/relational corpus substrate. Corpus Arbitration Layer [CAL] means the anchored corpus arbitration component. Rules Service means the first CAL deployment for rules systems. Role Service means the bounded role-behavior system. Agent Service means the later external-action layer.

This archive revises the AGE development vault so that acronyms, terms, and concepts are defined before use in the reader-facing documents. The Executive Summary and white papers are now standalone documents with local definitions, table-of-contents support, and architecture diagrams.
