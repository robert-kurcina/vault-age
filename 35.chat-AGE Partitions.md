# 1️⃣ Partitioning as a Scalability Lever

In AGE, you’re doing two things with partitioning:

1. **Spatial Partitioning** – the world is hierarchically divided (world → region → locus → submap).
    
2. **Narrative Partitioning** – story scope is hierarchically divided (spotlight → scene → act → chapter → arc → epic).
    

These serve several critical functions:

- **Bounding LLM context**: The LLM only ever sees the slice relevant to the active narrative focus. This is how you prevent drift.
    
- **Supporting modularity**: Each layer can be handled independently by LLMs, deterministic engines, or eventual multi-agent Roles.
    
- **Facilitating parallelism**: You can run multiple arcs, regions, or “scenes” in parallel without cross-contaminating state.
    

This is **directly transferrable** to interactive media or RaaS because it creates clear _domains of responsibility_ for AI or human actors.

# 2️⃣ From RPG to Interactive Media

- Optional static images

- **Dynamic interaction**: Partitioned narrative and world-state layers allow live, deterministic reactions without sending the entire world to the LLM.
    

Essentially:

> Partitioning becomes the “context manager” for multi-modal content generation.
